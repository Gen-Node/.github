Gennode turns idle machines into a network for science. Run a node and help power heavy bio & health computation — **DNA & genome analysis, protein folding, drug discovery and disease research** — while large, sensitive datasets are moved and stored **torrent-like: encrypted and sharded** across the network (planned). Contribution is measured as points (`uptime × capacity`), which convert to **`$GENNODE`** in a future airdrop.

### 🖥 Run a node
Download the desktop app for **macOS · Windows · Linux** from [Releases](https://github.com/Gen-Node/Gen-Node/releases), or run headless in one line:
```bash
npx @gennode/node
```

### 🔬 What the network computes
DNA & genome analysis · protein folding · drug discovery · disease research · multi-omics — the heaviest, most privacy-sensitive science.

### 🔒 Private by design
A torrent-like data layer (planned): bio datasets are encrypted **client-side**, then sharded across nodes — operators store pieces they **cannot read**. Decentralized *and* private.

### ⛓ $GENNODE
A planned **100% fair launch** on **Base** — no team allocation, no pre-mine. Rewards and the airdrop are funded by trading fees. *Not launched yet.*
